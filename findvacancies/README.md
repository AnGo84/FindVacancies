# FindVacancies

The module contains classes for parsing data from sites:
- [work.ua](https://www.work.ua/)
- [DOU.ua](https://dou.ua/)
- [grc.ua](https://grc.ua/)
- [djinni.co](https://djinni.co/)

### Technology stack:
- Framework: Spring boot 3
- Build: Maven

### **For work needs:**
- IDE
- JDK 17
- Maven
